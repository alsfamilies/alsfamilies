# ALS Families

我是一名 ALS（渐冻症）患者家属，也是一名工程师。我的家人于 2020 年确诊 ALS，目前已完全瘫痪，并接受了气管切开手术。

日常工作中，我主要运用人工智能与创新工程方法，开展 ALS 药物发现及发病机制研究。同时，我也在开发面向真实照护场景的实用工具，帮助更多渐冻症患者和家庭解决生活中的具体难题。

这项事业会长期以公益形式推进。相关工具将尽可能免费提供给患者、家属和照护者使用，并在真实反馈中持续迭代、优化和更新。

## 正在推进

- ALS 药物发现与发病机制研究
- 面向患者家庭的辅助沟通工具
- 低成本、可本地运行、重视隐私的照护辅助软件
- 基于真实照护场景的开源原型验证

## 当前项目

- [ALS 面部微动作 AAC](https://github.com/alsfamilies/als-facial-gesture-aac)：借助 ALS 患者仍可完成的少量面部微动作，向照护者及家人传递常用信息。

## 反馈

如果你是 ALS 患者、家属、照护者、医生、康复师、研究者或开发者，欢迎通过项目的 GitHub Issues 留下使用反馈、问题记录和改进建议。

## 使用边界

这些工具旨在为患者生活提供便利，但不能替代专业医疗治疗，也不能作为可完全依赖的紧急呼叫设备。请使用者结合具体照护环境自行评估使用风险。
